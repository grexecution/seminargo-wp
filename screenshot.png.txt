SCREENSHOT PLACEHOLDER
======================

WordPress themes require a screenshot.png file (1200x900px recommended).

To create a screenshot:

1. Activate the theme in WordPress
2. Visit your homepage
3. Take a screenshot (1200x900px)
4. Save as "screenshot.png" in this directory

Or create a simple one with this HTML template:

```html
<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            margin: 0;
            padding: 40px;
            background: linear-gradient(135deg, #2563eb, #1e40af);
            color: white;
            font-family: -apple-system, sans-serif;
            display: flex;
            align-items: center;
            justify-content: center;
            height: 900px;
            width: 1200px;
            text-align: center;
        }
        h1 { font-size: 72px; margin: 0 0 20px; }
        p { font-size: 24px; opacity: 0.9; }
    </style>
</head>
<body>
    <div>
        <h1>seminargo</h1>
        <p>Modern WordPress Theme</p>
        <p style="font-size: 18px; margin-top: 40px;">Clean • Fast • Responsive</p>
    </div>
</body>
</html>
```

Save this HTML, open in browser, screenshot it, and save as screenshot.png
